# ChumCards
(development name minecards)
Is a trading card plugin made for paper servers. It's heavily inspired by [BuddyCards](https://www.curseforge.com/minecraft/mc-mods/buddycards) one of my favorite mods of all time. So I watned to make my own for servers, there are many like it but this ones mine. I have many more plans and maybe even battling eventually. Check the wiki for instructions on how to install and make your own packs :).
